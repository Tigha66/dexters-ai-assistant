# Heartbeat checks - runs periodically to stay on top of tasks

## Daily Tasks (rotate through)

### Task 1: Check for responses
- Check email for replies from leads
- Check Telegram for new messages

### Task 2: Review pipeline
- Check crm/outreach-log.md for pending follow-ups
- Note any leads that need follow-up today

### Task 3: Research new leads
- Search for new plumbers in target cities
- Add new prospects to lead list

### Task 4: Social/media check
- Check if any landing pages got traffic
- Look for new reviews on competitor sites

### Task 5: Content ideas
- Note any interesting news in plumbing/AI space
- Think of outreach angles

---

## Notes
- Rotate through these tasks, doing 1-2 per heartbeat
- Don't check everything every time (avoid spam)
- Only alert human if there's something urgent
