üìå Platform-Specific Instructions

WordPress:

  1. Go to Appearance ‚Üí Theme Editor
  2. Click on header.php on the right
  3. Find <?php wp_head(); ?>
  4. Paste JSON-LD before that line
  5. Click Update File

Wix:

  1. Go to Settings ‚Üí SEO (Google)
  2. Scroll to Header Code
  3. Paste your JSON-LD code
  4. Click Save

Squarespace:

  1. Go to Settings ‚Üí Developer Options ‚Üí Code Injection
  2. Paste in Header section
  3. Click Save

Shopify:

  1. Go to Online Store ‚Üí Themes ‚Üí Actions ‚Üí Edit Code
  2. Click theme.liquid
  3. Find <head>
  4. Paste JSON-LD before </head>
  5. Click Save

Custom HTML/Other:

  1. Open your index.html file
  2. Find <head> at the top
  3. Paste JSON-LD anywhere between <head> and </head>
  4. Save and upload
