# Client Setup Checklist

Use this checklist when onboarding new clients to your AI automation service.

---

## Server Setup

- [ ] Hostinger VPS provisioned (Ubuntu 24.04)
- [ ] Security hardened
- [ ] UFW firewall (deny all, allow SSH)
- [ ] Fail2ban installed and active
- [ ] SSH key-only auth (passwords disabled)
- [ ] Tailscale installed + SSH locked to tailnet
- [ ] Config file permissions set (chmod 600)

---

## OpenClaw Setup

- [ ] OpenClaw installed and configured
- [ ] AI model connected (Claude API key)
- [ ] Gateway bound to loopback
- [ ] Tailscale Serve enabled for dashboard

---

## Messaging

- [ ] Telegram bot created via BotFather
- [ ] Bot token configured in OpenClaw
- [ ] DM pairing enabled for security
- [ ] Test message sent and received

---

## Customization

- [ ] SOUL.md customized for their business
- [ ] AGENTS.md configured with memory system
- [ ] Custom prompts for their industry

---

## Automation

- [ ] Morning briefing (daily)
- [ ] CRM review (daily)
- [ ] Lead research (nightly)
- [ ] Lead outreach (nightly)
- [ ] Follow-up automation

---

## Client Handoff

- [ ] Documentation sent to client
- [ ] 30-minute walkthrough call completed
- [ ] Monthly retainer agreement signed
- [ ] First month invoice sent
